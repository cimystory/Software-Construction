This is a social media analyser framework with one data plugin and two analyser plugins implemented.

How to use : 

- Write the Query in the textbox and enter proceed. Wait for some time while the button Proceed is highlighted. Then click the screen (JFrame) and you shall see the analyses. Change the tab to have another plugin working which is doing another analyses.

- You can search for another Query and click on the screen and the screen will be refreshed.

Extensions :

The framework is capable of installing any analyser/data plugin.

Important Instructions :

For your Analyses plugin to work, you have to have an object of class DataManager defined for you already from whom you can get all the data you require. It is the data expert speaking on behalf of framework.
